                                        • HTTP接口概述

                                        1、接口简述

                                        服务端需要开发HTTP/HTTPS的接口供使用方请求访问，用于使用方主动向服务端发送数据或者请求数据。本质上就是前端页面（设备）请求服务端的接口，通信数据格式是Json。

                                        2、服务端接口规范

                                        接口地址： http://服务器ip:port/ 或 https://服务器ip:port/（IP可替换成已备案的域名）

                                        接口形式： 通过HTTP/HTTPS请求的方式对外提供服务。

                                        接口返回： 所有接口返回的数据，都包含code，msg字段

                                        返回格式： json

                                        3、服务端接口返回示例

                                        {
                                            "code": 200,      
                                            "Msg": "success"
                                        }

                                        上传识别成功记录

                                        简要描述：

                                        • 设备识别成功后通过这个接口上传人脸识别的记录信息到服务器。以下 2 种场景识别成功不会调用该接口，会调用上传陌生人记录接口： ① 设备设置-【识别设置】里，可通行人员为任何人时，任何人的通行记录会调用上传陌生人记录接口。 ② 如果是设备单机录入的（在设备人员管理页面点击底部 + 号录入的）用户 id 为 DL 开头的用户识别成功会调用上传陌生人记录接口。
                                        • 特别注意：服务端响应的 json 中 Result 不为 0 时，设备会当做记录上传失败了，此时记录会存储在本地，并且每间隔两分钟调用一次上传记录接口，上传本地存储的离线记录，只有服务端返回 Result 为 0 时才会删除离线记录。
                                        • 服务端成功接收数据后务必要把响应中的 Result 返回为 0 ，避免设备重复调用记录上传接口。

                                        请求URL：/api/v1/record/face

                                        请求方式：POST

                                        参数： (格式：application/json ，内容放 body)

                                        设备端请求示例

                                        xxxxxxxxxx
                                        { 
                                            "sn":"xxxx",
                                            "Count":1,
                                            "logs":[
                                                {
                                                    "user_id":"11",
                                                    "recog_time":"2019-01-01 11:11:11",
                                                    "recog_type":"face",
                                                    "photo":"base64",
                                                    "body_temperature":"36.5",
                                                    "confidence":"95.5",
                                                    "reflectivity":86,
                                                    "room_temperature":25.5
                                                }
                                            ]
                                        }

                                        服务端返回示例

                                        xxxxxxxxxx
                                        {
                                            "Result": 0,
                                            "Content": "",
                                            "Msg": ""
                                        }
